# Codlexa

Codlexa is a futuristic WebGL platform focused on holographic web experiences.

Features:

• WebGL powered interface  
• 3D hologram interaction  
• Gesture control (coming soon)  
• API system for businesses  
• Hologram studio playground  

Tech Stack:

- Three.js
- WebGL
- Supabase
- JavaScript
- Netlify deployment

Purpose:

Codlexa aims to enable developers and businesses to integrate holographic
3D interactions into their websites using powerful APIs.
